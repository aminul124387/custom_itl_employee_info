Employees Competences Views

"Finally you can view and search employees' education, experience, and skills in a separate tree views"

Odoopedia

Description

This module adds a valuable addition to the existing functionality of the Odoo HR module. By displaying the employee's education and experience in separate tree views, a more comprehensive views of an employee's qualifications, skills and work history will be provided.
